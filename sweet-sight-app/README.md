# Sweet Sight App 🍰

A delightful dessert gallery web application that allows users to browse and add their favorite desserts.

## Features

- **Interactive Gallery**: Browse through a collection of beautiful desserts
- **Category Filtering**: Filter desserts by categories (Cakes, Cookies, Pastries)
- **Add New Desserts**: Submit your favorite desserts with descriptions
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Beautiful UI**: Modern gradient design with smooth animations

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript (ES6+)

## How to Use

1. Open `index.html` in your web browser
2. Browse through the dessert gallery
3. Use the filter buttons to view specific categories
4. Fill out the form to add your own favorite desserts
5. Watch your dessert appear in the gallery!
